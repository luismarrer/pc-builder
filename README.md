# PC Builder

## Descripción

PC Builder es una aplicación web que permite a los usuarios construir su propio PC eligiendo los componentes que desean.

Nace como parte de las tares necesarias para completar el curso de Frontend con HTML, CSS y JS de Mastermind.ac.

Las notas sobre del profesor sobre el proyecto:

```md
# Notas

## ¡RETO!

Teniendo en cuanta lo visto en las clases, implementaremos mediante HTML, CSS y JS un carrito de compra para los componentes de tu próximo PC soñado.

Tendremos que realizar los siguientes pasos:

1. Diseñar una página con HTML y CSS donde tengamos tres zonas claramente diferenciadas:
    - Parte izquierda con un listado de componentes que se pueden usar.
    - Parte derecha un contenedor donde arrastrar los componentes y que sería nuestro cesta de la compra.
    - Suma total del carrito.
El procedimiento de uso sería:
    - El usuario va arrastrando los elementos de la parte izquierda al carrito.
    - Cuando se añade al carrito se suma a la compra total el precio del nuevo componente añadido.
    - El usuario puede quitar elementos de la cesta de la compra directamente con un  botón o arrastrando el componente de nuevo a su origen.

Por lo tanto tendremos que usar como mínimo el Web API Drag and Drop además de las tecnologías utilizadas hasta ahora. 

### Mega reto

Puedes plantear muchas modificaciones a partir del planteamiento inicial:

1. Almacenar en el localStorage el carrito actual para poderlo recuperar una vez se vuelva a la página preguntando al usuario si quiere recuperarlo.
2. Sustituir los componentes en formato texto por imágenes.
3. Incluir incompatibilidades entre componentes, como por ejemplo entre placas y procesadores.

¡ÁNIMO!
```

## Corrección

```md
Enhorabuena por el entregable que cumple funcionalmente y visualmente con el planificador de componentes para tu PC: - Código de etiquetas HTML bien formado. - Selectores, propiedades y valores correctos en el CSS. - La estructura de directorios y de ficheros es la correcta. - Se activa correctamente el dragStart. dragOver y drop funcionando el drag&drop. - Cálculo correcto a través de JS de los elementos añadidos al cálculo de precio final. - Visualmente se añaden imágenes y elementos que hacen fácil la interacción en el proyecto. - El uso de comentarios también es de agradecer, son siempre necesarios tanto para otro desarrollador que trabaje contigo como para ti mismo si el programa comienza a ser más complejo. ;) Extra point con la posibilidad de eliminar elementos del cálculo. Espero poder ver tus próximos entregables, seguro que son del mismo nivel que este.
```
